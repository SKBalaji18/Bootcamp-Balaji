Static Site Generation - Exercise Notes

Process

The task was to design and prototype a static version of an e-commerce website like Amazon or Flipkart. The solution involved static site generation, ensuring scalability and hosting viability on platforms like Netlify or Cloudflare. The catalog data was retrieved from a `data.json` file, which was used to display the items without the need for server-side processing.

The project included implementing client-side search with features. Category filtering was also added to improve usability. Optimization techniques were explored to manage large catalogs while considering browser limitations.

Challenges

Challenges included implementing auto-suggestions in the search functionality and refining category filtering for better user experience.
