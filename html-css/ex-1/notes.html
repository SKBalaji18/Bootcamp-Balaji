HTML Elements and Structure Notes

Semantic HTML Elements

We used several semantic HTML elements like <header>, <footer>, <main>, <article>, and <section> to organize the page's content logically. These elements improve accessibility and SEO.

HTML Structure and Layout

The golden layout structure was achieved using <nav> for sidebars, <main> for the main content, and <article> and <aside> to differentiate content types.

Accessibility Considerations

We used clear table headers for accessibility and included alt text for images (if any). Proper nesting and alignment of elements were also maintained.

Challenges

  • Ensuring proper nesting of elements and structure.
  • Handling layout and design using HTML alone without heavy CSS frameworks.
  • Making sure all elements are properly accessible.
