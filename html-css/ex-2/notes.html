CSS Techniques Used

CSS Variables

We defined CSS variables for colors, font sizes, and spacing in the variable-n.css file. These variables are referenced throughout the main stylesheet to ensure consistent styling across the webpage.

Default Values for Variables

We used the var(--variable-name, default-value) syntax to provide default values for variables when they are not defined elsewhere.

Alternate Row Styling

The nth-child selector was used to style alternate rows of a table to improve readability.

Dynamic Theme Switcher

The dynamic theme switcher uses CSS variables for the background, text color, and primary/secondary colors. JavaScript is used to toggle between light and dark modes by modifying the data-theme attribute on the HTML element.

Challenges

  • Ensuring that CSS variables were properly applied throughout the stylesheet.
  • Debugging the theme switcher functionality to ensure it correctly toggles the theme.
