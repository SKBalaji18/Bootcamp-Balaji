Level 2

Thought Process

In this level, I extended the functionality of the logger to include file transport, enabling logs to be written to a file. I also customized the log format to include timestamps for better traceability. The implementation was straightforward because I already had a basic understanding of winston from the previous level. Adding file transport involved specifying the output file in the logger configuration, while formatting with timestamps required using winston.format.combine and winston.format.timestamp. This level reinforced my understanding of winston and its features.
