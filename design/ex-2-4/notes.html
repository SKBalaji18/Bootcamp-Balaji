Level 1: Adding and Styling Images

For Level 1, I added three images to the web page using a placeholder image service (e.g., Lorem Picsum). CSS was applied to set the width, height, and border-radius of the images, giving them a uniform appearance while maintaining flexibility for different designs.

The main challenge was selecting appropriate placeholder images that fit the intended layout and ensuring the styles were responsive across devices.

Level 2: Image Editing and Overlays

In Level 2, I edited an image to adjust its size and compression, ensuring faster load times without compromising quality. A full-bleed image was added as a page header, styled using CSS to cover the entire width of the page and create a striking visual impact.

CSS overlays, such as gradients and color tints, were applied to the header image to enhance text readability and aesthetic appeal. Balancing the overlay opacity and ensuring proper contrast posed a minor challenge, especially when viewed on different devices.

Level 3: Responsive and Optimized Images

For Level 3, I implemented responsive images using the <picture> element, which allowed the browser to load appropriate image sizes based on the viewport. This approach ensured optimal performance and clarity on devices with varying screen resolutions.

I created an image with high-resolution and low-resolution versions, dynamically displayed depending on the user’s device. Lazy loading techniques were also employed to defer the loading of images until they were in the viewport, reducing initial page load times.

The primary challenges included ensuring seamless transitions between image resolutions and verifying that lazy loading didn’t affect user experience negatively on slower networks.

This project offered insights into modern image handling techniques for web design, emphasizing responsiveness, performance optimization, and visual appeal. Key takeaways include leveraging placeholder services, CSS overlays, and advanced features like the <picture> element and lazy loading.
