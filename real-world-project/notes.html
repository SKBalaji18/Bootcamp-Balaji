Disease Explorer

Process

Developed an interactive widget to search, explore, and select diseases using React.js, JavaScript, and CSS. The widget retrieves hierarchical disease data from EFO or MONDO using the EBI OLS API and displays the hierarchy for user exploration. It allows users to select a disease and fetch its code.

Challenges Faced

Faced issues while fetching data from the API. Debugged errors by inspecting API responses in the console and resolved them to retrieve the disease hierarchy successfully.
