Using Tools - Exercise Notes

Process

The exercise involved setting up a TypeScript project with a package manager (npm) and configuring it using a tsconfig.json file. The project focused on solving three problems:

  • Problem 1: Load a JSON file, render a table dynamically, and include sorting functionality.
  • Problem 2: Fetch data from a CSV file using PapaParse, enhance the table with column filtering and row highlighting.
  • Problem 3: Integrate a build tool (e.g., tsc for transpilation) and package the project for both development and production environments.

Vite was used for the development server, live reload, linting, and production build bundling.

Challenges

I encountered many errors as I am new to TypeScript and Vite. To resolve these, I referred to documentation on StackOverflow and official pages to understand and fix the issues.
