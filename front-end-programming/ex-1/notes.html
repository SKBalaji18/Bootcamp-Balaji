Process

The project started with a single-file application using HTML, CSS, and JavaScript. A dynamic student table was created with options to add rows, highlight them, and change the font using a Google Font.

JavaScript was modularized into two files: data-loader.js for loading JSON data and table-renderer.js for rendering the table. Later, AG Grid was integrated with data sourced from a CSV file, enhanced with sorting, filtering, and row highlighting features.
